## __Introduction__
Welcome to the Cloudscape repository. In here you'll find this README and 3 directories:
- `lessons`
- `blog-project`
- `study-app-project`

The `lessons` directory is where you will find the associated code and instructions relevant to each lesson in the course.

The `blog-project` directory is where you will find the associated code and instructions for the Blog website project we will be building.

The `study-app-project` directory is where you will find the associated code and instructions for the Study App web application project we will be building.

## __Description__
In this repository, you will find code examples, boiler plate code, instructions, lists of which technologies are being used, links to documentation, and much more.
